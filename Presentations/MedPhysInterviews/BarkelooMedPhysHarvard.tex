\frame{\frametitle{Conclusions}
\begin{itemize}
\item I have created model independent signal samples to search for new physics with flavor changing neutral current decays in top pair events
\item Developed and implemented a neural network for signal classification
\item Currently working to ensure well modeled backgrounds and account for systematic errors
\item After unblinding if I find any excess in signal data events it is a strong indication of physics beyond the Standard Model
\end{itemize}
